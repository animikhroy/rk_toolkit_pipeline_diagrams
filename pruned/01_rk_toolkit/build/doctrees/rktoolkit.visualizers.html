class rktoolkit.visualizers.visualizer.RKDiagram(rkmodel: rktoolkit.models.graph.RKModel, placed_nodes, links)¶

Bases: object

An RKDiagram is the manifestation of an RK-Model

It contains the positions of each node, information about how to represent such nodes, as well as edge visualizations.

It is purely a visualization diagram. To work effectively, the visualized space must be in nD < 4
